# tflearn_mnist
tflearn on mnist datasets

## Supported networks:
* LeNet ([ref](http://eblearn.sourceforge.net/beginner_tutorial2_train.html))
  * conv (5x5), l2_pooling (2x2)
  * conv (6x6), l2_pooling (2x2)
  * conv (6x6), l2_pooling (2x2)
  * conv (6x6)
  * fully-connected
* Default
 * full-connected, 20 , ReLU
 * full-connected, 10, softmax
